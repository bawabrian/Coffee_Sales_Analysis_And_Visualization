Coffee Sales Dashboard (Excel + SQL)


Project Overview
This project focuses on analyzing coffee sales data and presenting insights through an interactive Excel dashboard. The aim was to use SQL for data cleaning, preparation, and calculations, and Excel for visualization using Pivot Tables and interactive controls.
The dashboard allows users to explore sales performance over time, across countries, and by product and customer attributes.

Objective
The objectives of this project are to:
- Prepare and structure sales data using SQL
- Build an interactive dashboard using Microsoft Excel
- Analyze sales trends, customer performance, and product breakdowns
- Practice an end-to-end analytics workflow from raw data to insights

Tools Used
- SQL (for data cleaning, transformation, and calculations)
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Timeline
  - Slicers
  - Dashboard layout and formatting

Dashboard Summary

Interactivity
The Excel dashboard includes:
- A timeline for filtering data by order date
- Slicers for:
  - Roast Type (Dark, Medium, Light)
  - Product Size (0.2 KG, 0.5 KG, 1 KG, 2.5 KG)
  - Loyalty Card status (Yes / No)

All visuals update dynamically based on the selected filters.

Key Visuals & Insights

Total Sales Over Time
- Shows sales trends across multiple years
- Breaks down performance by coffee type
- Helps identify growth patterns and seasonality

Sales by Country
- Compares total sales across different countries
- Highlights top-performing markets

Top 5 Customers
- Identifies the highest-value customers by total sales
- Useful for understanding customer concentration and loyalty

Data Notes
- SQL was used to clean, enrich, and prepare the dataset prior to analysis
- Excel was used exclusively for visualization through Pivot Tables and charts
- The dataset is used for learning and demonstration purposes
Purpose of the Project
This project was created to:
- Strengthen SQL-based data preparation skills
- Improve Excel dashboard design using Pivot Tables
- Practice translating data into clear, business-focused insights
- Build practical analytics projects for GitHub
